第零章——建站教程

参考教程：MacOs使用hugo、github搭建个人博客&&使用Hugo搭建个人博客Mac+Docker

  • GitHub 官方使用建议和限制，在使用的时候项目和网站的大小不要超过 1GB，也不要过于频繁的更新网站的内容（每小时不超过 10 个版本），每个月的也要注意带宽使用上限为 100GB。
    • 中小型博客或项目主页的最佳选项之一。
  • Hugo 依靠Go语言进行开发，生成静态html非常快
    • 支持Markdown
    • 生成的页面速度为秒级
    • 生成的静态文件可以托管在各大平台（绑定域名即可访问）

一、准备工作

  • 1、在个人主界面里选择创建一个新的 Repository

    Repository name 的位置填写域名，格式是 username.GitHub.io

    官方说如果你的username中有大写字母，在创建库的时候必须小写，不过实际上没什么影响。

  • 2、安装brew

安装brew

  • 官网命令

    /bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    报错：fatal: unable to access ‘https://github.com/Homebrew/brew/': SSL: no alternative certificate subject name matches target host name ‘github.com’ Failed during: git fetch –force origin

  • 其他教程命令

    • /usr/bin/ruby -e “$(curl -fsSL https://cdn.jsdelivr.net/gh/ineo6/homebrew-install/install)"

      报错：curl: (7) Failed to connect to cdn.jsdelivr.net port 443: Operation timed out

    • /bin/zsh -c “$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

      是一个安装脚本，亲测有效

      参考文章：https://blog.csdn.net/m0_48895571/article/details/123897554

二、Mac安装hugo

$ brew install hugo

hugo version可查看是否已经安装好

创建站点

1

 $ hugo new site myblog（自定义名称）

自定义名称为在当前目录下生成文件夹名，也是你的网站的大本营，后续大部分的操作都在这个文件夹下进行，此目录在下文中我统称为根目录。

运行

1
2

$ cd myblog
$ hugo server

三、安装Hugo主题

  • 可在hugo官网主题进行下载主题

    可以使用git命令对主题进行克隆，注意，下载的主题要放在博客根目录的themes目录下。

    在根目录下执行命令

    hugo server -t hugo-tranquilpeak-theme --buildDrafts

    其中hugo-tranquilpeak-theme为下载的主题名称,，成功会返回localhost:1313

    在浏览器中输入:localhost:1313,如果能正常显示，说明在本地服务器运行已经没问题。

  • 选用别人做的主题

    选好中意的主题了，那么就一定要看作者的配置介绍

    这个配置介绍一般都是介绍如何配置config.toml，但这个介绍可能不够全，

    那么就要自己看clone下来的主题中这个文件夹exampleSite中的config.toml。

    有的主题还有full-config.toml,这个内容更全面。

    以我使用的主题为例，路径结构为：themes\hello-friend-ng\exampleSite\config.toml

    参考博客_详细解读配置内容

    $ cd myblog
$ git clone https://github.com/flysnow-org/maupassant-hugo themes/maupassant

    报错：fatal: unable to access ‘https://github.com/flysnow-org/maupassant-hugo/': SSL: no alternative certificate subject name matches target host name ‘github.com’

    找了半天没找到解答，多试着clone几次就好了

    • baseurl：配置站点基址，如果要放在github上或者自己的云服务器上，请正确配置地址，否则出了能访问主页，其他地方都是 404
